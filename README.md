# Holy Quest AI - AI Assistant Extension

AI-powered development assistant with conversation memory and token tracking.

## Features

- 💬 **Conversation Memory** - Maintains context across chat sessions
- 📊 **Token Tracking** - Visual indicator showing token usage (200k max)
- 📝 **Auto-Summary** - Generate context transfer summaries for new sessions
- 🤖 **Multi-Agent System** - Specialized agents for different tasks
- 📎 **Image Support** - Attach screenshots for visual debugging
- ✨ **Markdown Rendering** - Beautiful code formatting with copy buttons

## Setup

1. Install the extension
2. Set your Anthropic API key in settings
3. Open the Holy Quest AI sidebar
4. Start chatting!

## Usage

Type your questions in the chat box. The extension will:
- Remember conversation history
- Track token usage with visual bar
- Suggest summaries when approaching token limits
- Format responses with syntax highlighting

## Commands

- Click **📝 Summary** to generate context transfer document
- Click **🗑️** to clear conversation history
- Use **Ctrl+Enter** to send messages

## Requirements

- VS Code 1.85.0 or higher
- Anthropic API key

## Extension Settings

This extension contributes the following settings:

* `holyQuestAI.apiKey`: Your Anthropic API key
